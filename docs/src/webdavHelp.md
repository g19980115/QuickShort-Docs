# WebDav 配置教程

**所有支持WebDav的云盘都可以，建议使用坚果云，此处教程以坚果云举例.**

1. 手机下载安装**坚果云**APP <br>
2. 注册/登录坚果云<br>
3. 点击左上角菜单-设置-第三方应用管理<br>
4. 在第三方应用管理中添加应用<br>
5. 将服务器地址、账户，还有应用密码填写到快捷方式app的WebDav设置中<br>
6. App中的子文件夹即为存储目录，支持多级目录<br>
7. 配置完成后手动执行一次备份到WebDav，测试是否成功<br>
8. WebDav同一天的备份会覆盖，不同日期的备份不会覆盖<br>

**错误码** <br>
1.(401)账号或密码错误，注意：是应用密码，不是登录密码
